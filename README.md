# ToDoApplicationProject
Приложение для списка дел. 
Функции Добавления, Редактирования, Удаления задач.
Язык разработки C# 
Используемые технологии WPF.
Среда разработки  Microsoft Visual studio.
База данных Microsoft SQL Server.
